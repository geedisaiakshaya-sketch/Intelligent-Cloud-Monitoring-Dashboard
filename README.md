# Intelligent Cloud Monitoring Dashboard

## Overview

The Intelligent Cloud Monitoring Dashboard is a cloud-based monitoring solution developed using AWS services. The project provides a web dashboard that displays infrastructure metrics such as CPU usage, memory usage, network utilization, system health, and cloud resource information. AWS services are used for backend processing, data storage, and cloud monitoring.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- AWS EC2
- AWS Lambda
- Amazon API Gateway
- Amazon DynamoDB
- Amazon CloudWatch
- IAM
- Git & GitHub

## Features

- User-friendly monitoring dashboard
- CPU usage monitoring
- Memory usage monitoring
- Network traffic monitoring
- System health status
- Cloud resource monitoring
- Cost monitoring
- AWS serverless backend
- Responsive web interface
