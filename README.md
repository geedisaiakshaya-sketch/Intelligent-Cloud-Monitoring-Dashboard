# Intelligent Cloud Monitoring Dashboard

## Overview

The Intelligent Cloud Monitoring Dashboard is a serverless cloud monitoring application built using AWS services. It collects infrastructure metrics, processes them using AWS Lambda, stores data in Amazon DynamoDB, and displays the results through a web dashboard.

## Technologies Used

* AWS Lambda (Python)
* Amazon DynamoDB
* Amazon API Gateway
* HTML
* CSS
* JavaScript

## Features

* CPU Usage Monitoring
* Memory Usage Monitoring
* Network Usage Monitoring
* Cost Monitoring
* System Health Status
* Serverless Cloud Architecture

## Architecture

Dashboard (HTML/CSS/JS)
↓
API Gateway
↓
AWS Lambda (Python)
↓
Amazon DynamoDB

## AWS Services Used

1. AWS Lambda
2. Amazon DynamoDB
3. Amazon API Gateway

## Project Outcome

This project demonstrates a cloud-native monitoring solution using AWS serverless services. The dashboard provides real-time visibility into infrastructure metrics through a simple and user-friendly interface.
